# Артемида-2: Симуляция траектории полёта

Проект для моделирования траектории свободного возврата космического корабля «Орион» в рамках миссии Artemis II. Проект состоит из физического движка на C++ и визуализации телеметрии на Python.

## Архитектура и стек технологий

* **C++:** Отвечает за парсинг эфемерид и математическое моделирование полета методом численного интегрирования.
* **Python:** Скрипты на базе `Pandas` и `Matplotlib` для отрисовки двойной анимации.
* **Batch:** Скрипт автоматизации для связки модулей и запуска проекта в один клик.

## Математическая модель

В основе симуляции лежит решение задачи трех тел (Земля, Луна, Орион) с использованием **Метода Рунге-Кутты 4-го порядка**. 

**Сравнение симуляции с реальными данными NASA:**
Уникальной особенностью проекта является параллельное сравнение траекторий. Расчетная траектория выводится на экран синхронно с реальной телеметрией NASA, что позволяет визуально оценить расхождения идеализированной физической модели по сравнению с реальным N-body моделированием. 

Погрешность в расчетах возникает в силу того, что данная модель не берет во внимание гравитационное влияние других тел Солнечной системы, давление солнечного света, а также микрокорректировки траектории посредством включения двигателей Ориона на маршевом участке.

## Как запустить локально

**Требования:**
* **Visual Studio 2022** (или другой компилятор C++)
* **Python 3** с библиотеками `pandas`, `numpy`, `matplotlib`

После клонирования данного репозитория и сборки C++ проекта, для запуска требуется только открыть файл `start.bat`. Скрипт сам запустит сначала C++ код, а затем на основе сгенерированных им данных откроет Python-визуализацию.

## Источники данных
Реальные эфемериды для Луны и корабля «Орион» получены из официальной базы данных **NASA JPL Horizons**.
